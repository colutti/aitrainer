# Sobre o projeto:

- Esse e um software que simula um personal trainer virtual, ajudando usuarios a criar e seguir planos de exercicios personalizados com base em seus objetivos de fitness, nivel de experiencia e preferencias usando AI.
- Tem uma parte de frontend feita em Angular (frontend) e uma parte de backend feita em Python com FastAPI (backend).
- A ideia e que o usuario possa interagir com o personal trainer virtual atraves de uma interface web, onde ele podera inserir suas informacoes pessoais, objetivos de fitness e preferencias de exercicios e tambem fazer perguntas sobre exercicios, nutriciao e saude em geral.
- Usa uma base de dados MongoDB para armazenar as informacoes dos usuarios, planos de exercicios e historico de atividades.
- Usa Mem0 para guardar o contexto das conversas dos usuarios com o personal trainer virtual. Essas sao memorias de longo prazo que ajudam a personalizar as respostas do personal trainer virtual com base no historico do usuario.
- Permite que o usuario personalize seu treinador virtual escolhendo entre diferentes estilos de comunicacao, como motivacional, tecnico ou amigavel.
- Permite que o usuario informe seus dados pessoais, como idade, peso, altura e nivel de experiencia em fitness, para criar planos de exercicios mais precisos.
- Quando voce executa o backend e frontend localmente para testes, voce faz isso em segundo plano para nao bloquear seu fluxo de trabalho. ISSO E ESSENCIAL.
