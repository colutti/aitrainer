# Politica de Privacidade - FityQ

Ultima atualizacao: 2026-04-01

## 1. Objetivo

Esta politica explica como o FityQ coleta, usa e protege dados pessoais dos usuarios.

## 2. Dados que podem ser coletados

1. Dados de cadastro (ex.: nome e email).
2. Dados de uso da plataforma (ex.: interacoes com funcionalidades).
3. Dados informados voluntariamente para personalizacao de experiencia.

## 3. Finalidades de uso

Usamos dados para:

1. Autenticacao e seguranca da conta.
2. Operacao e melhoria do servico.
3. Personalizacao de funcionalidades.
4. Cumprimento de obrigacoes legais e regulatórias, quando aplicavel.

## 4. Compartilhamento

Os dados podem ser compartilhados com provedores de infraestrutura e servicos essenciais (ex.: autenticacao, processamento e armazenamento), sempre dentro de necessidade operacional e com controles de seguranca.

## 5. Seguranca da informacao

Adotamos medidas tecnicas e organizacionais para proteger os dados contra acesso nao autorizado, perda ou uso indevido.

### Status de criptografia ponta a ponta (E2EE)

O FityQ utiliza criptografia ponta a ponta (E2EE) nos fluxos aplicaveis, como parte das medidas de seguranca da informacao e protecao de dados.

## 6. Retencao

Os dados sao mantidos pelo tempo necessario para cumprir as finalidades descritas nesta politica e obrigacoes legais.

## 7. Direitos do titular

Quando aplicavel, o usuario pode solicitar acesso, correcao, atualizacao ou exclusao de dados, conforme legislacao vigente.

## 8. Alteracoes desta politica

Podemos atualizar esta politica periodicamente. A versao vigente sera publicada com a data de atualizacao.

## 9. Contato

Para duvidas sobre privacidade e dados, use os canais oficiais de suporte do FityQ.
